VisualVNC is a connector between virtual network and physical network. A virtual network mainly refers to the topology created by GNS3 (Graphical Network Simulator, a popular network simulator), the physical network means the real machines or virtual machines (created by VMware, virtualbox and so on) which are running Windows. What VisualVNC achieves is that 



VisualVNC changes the Windows machine's routing behavior via a NDIS intermediate driver,   VNC is based on NDIS driver technique and limited to Windows platform.
